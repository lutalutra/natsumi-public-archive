# PRJ0031 — OpenDNA Museum Collaboration Ludwig Museum Exhib

## What goes here
- `images/raw/` = original source files dumped from Wix / Dropbox / camera. Keep original filenames when possible.
- `images/selected/` = working shortlist copied from raw. Keep filenames or add notes if needed.
- `images/public/` = final web-ready files with fixed names: `cover.jpg`, `thumb.jpg`, `gallery_01.jpg`, etc.
- `docs/` = PDF, press, installation plan, grant documents, scans.
- `text/` = statement, captions, Japanese / English descriptions.
- `video/` = trailers, screen recordings, documentation clips.
- `audio/` = sound materials if any.
- `exports/` = temporary exports for applications or website output.
